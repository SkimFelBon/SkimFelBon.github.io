Smoke testing

Task: Test main features of application

  • login into system
  • create new board
  • add new card
  • add text to title
  • add text to description
  • save card
  • refresh page
  • check if text in title is the same as expected
  • check if description is the same as expected

Functional testing

Task: Test attach file feature of trello app

Expected result: the files should be the same

  • login into system
  • navigate to existing board
  • click on attach button
  • upload file
  • refresh page
  • download file
  • compare files

Stress testing

Task: Server should allow up to 500 concurrent connections.

Procedure: emulate amount of requests to server greater than pick value, for instance, using specific software

Expected result: user data should't be lost even in such situation

Compatibility testing

Task: Test improper operation on a supported platform. Test it using Chrome, Edge, Firefox etc

Test Steps: run previous tests using different browsers
